### About Me

I'm a technical professional with 10+ years across software engineering, SaaS platform support, and data.
Most recently I worked at Dimagi on CommCare, supporting enterprise deployments for global health organizations: troubleshooting APIs and integrations, digging through logs and SQL, and turning messy problems into clear fixes.
Before that I spent 6+ years building and shipping web and mobile applications with PHP (Laravel), Python (Django), and React.

### What I work with

- **Backend & data:** Python, SQL, PHP/Laravel, PostgreSQL, MySQL, REST APIs
- **Analytics:** Power BI, reporting pipelines, dashboards
- **Automation:** Zapier workflows, Python scripting, ticket routing automation
- **Platforms & tools:** CommCare, Salesforce, Jira, Zendesk, Docker, Linux, AWS (AI Practitioner certified)
- **Certifications:** PMP, AWS AI Practitioner

### What I care about

I like working where engineering meets real users.
Reproducing a tricky bug, automating a workflow that saves a team hours every week, or building a dashboard that finally makes the data make sense.
